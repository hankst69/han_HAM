Welcome to the SDRplay Start Here Page

RSP Newcomers: If you are completely new to SDRs then we recommend:

  • Watching the SDRconnect introductory video: https://youtu.be/kHOVAJi1JBs
  • Reading the SDRconnect “Quick Start” guide – you can click “Translate” for your own language version: SDRconnect Quick Start guide – SDRplay    (you can also download an English version in pdf format here:  https://www.sdrplay.com/docs/SDRconnectBasics.pdf
  • Downloading and Installing the SDRconnect software from here: https://www.sdrplay.com/sdrconnect/
  • Start “playing”

Networked RSP users: Please go to our dedicated nRSP-ST start page: https://www.sdrplay.com/nRSPstart/

Please note: SDRconnect works on Windows, MacOS and Linux machines, and is at the heart of our core development efforts – it is a replacement for our older (Windows only) SDRuno software which includes features not yet available on SDRconnect (e.g. rig control and scanner). 

RSP Advanced users: Depending on what you want to use your RSP for, go to our SDR software page where you will find links to SDRuno for Windows, legacy software, popular 3rd party software and our multiplatform API software: https://www.sdrplay.com/downloads/
